# 🖥️ PC-Cloud
> Ubuntu en la nube con un solo click, gratis y sin instalar nada!

---

## 🚀 Cómo usar
1. Click en el botón verde **Code**
2. Click en **Codespaces**
3. Click en **Create codespace on main**
4. Espera ~2 minutos que configure todo solo
5. Se abrirá el puerto 3000 automáticamente

---

## ✅ Qué incluye
- 🖥️ Ubuntu XFCE completo en el navegador
- 🌐 Chromium preinstalado en español
- 📁 Gestor de archivos Thunar
- 📝 Bloc de notas Mousepad
- ⏰ Keepalive automático (no se apaga solo)
- 🔄 Auto-restart si se cae
- 🐍 Python3, pip
- 🟩 Node.js, npm
- 📦 wget, curl, git, nano, htop, neofetch
- 🗜️ zip, unzip
