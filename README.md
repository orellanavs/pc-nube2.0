# 🖥️ PC-Cloud
> Ubuntu en la nube con un solo click, gratis y sin instalar nada!

---

## 🚀 Cómo usar

1. Click en el botón verde **Code**
2. Click en **Codespaces**
3. Click en **Create codespace on main**
4. Espera ~2 minutos que configure todo solo
5. Se abrirá el puerto 3000 automáticamente
6. Usuario: `forgex` / Contraseña: `pc-nube`

---

## ✅ Qué incluye

- 🖥️ Ubuntu XFCE completo en el navegador
- 🎮 RetroArch con cores GBA, DS y 3DS
- 🖼️ Wallpaper personalizado
- ⏰ Keepalive automático (no se apaga solo)
- 🐳 Auto-restart si se cae
- 📦 curl, wget, git, nano, htop, neofetch

---

## 🎮 Correr un juego

Descarga tu ROM y córrela desde la terminal del Ubuntu:

**GBA:**
```bash
retroarch -L /config/.config/retroarch/cores/mgba_libretro.so ~/Downloads/juego.gba
```
**DS:**
```bash
retroarch -L /config/.config/retroarch/cores/melonds_libretro.so ~/Downloads/juego.nds
```
**3DS:**
```bash
retroarch -L /config/.config/retroarch/cores/citra_libretro.so ~/Downloads/juego.3ds
```
